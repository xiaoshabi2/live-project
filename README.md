### <center> 项目介绍 </center>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;本项目采用前后端分离技术开发，前端使用boostrap+ajax等框架，后端采用Springboot+Spring+SpringMVC+Mybatis等框架进行开发，采用了Python进行数据挖掘和分析。
<br/>
本项目前端和后端都部署到了服务器上，并用了域名进行解析。
